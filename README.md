# 悦动云

悦动云是一个面向在线健身、训练陪伴与互动社区场景的前后端分离微服务项目，围绕训练内容管理、训练计划、跟练学习、互动问答、积分成长、营销活动、订单支付、用户中心与后台运营等核心业务，提供从用户侧到管理侧的一体化解决方案。

项目聚焦“内容训练 + 陪伴学习 + 社区互动 + 平台运营”的业务闭环，适合作为健身类内容平台、训练营平台与中后台业务系统的实践项目。

## 项目定位

- 面向在线健身与训练成长场景的综合平台
- 支持训练内容展示、购买、学习、记录、互动、激励的完整链路
- 支持后台对训练、教练、用户、订单、营销等业务进行统一管理

## 核心功能

### 用户端

- 首页训练内容推荐
- 训练分类浏览与搜索
- 训练详情展示
- 跟练中心与视频学习

- 训练计划管理
- 训练笔记记录
- 互动问答与回复
- 积分、签到与排行榜
- 优惠券领取、兑换与使用
- 购物车、结算、支付、订单管理
- 个人中心与学习数据查看

### 管理端

- 训练内容管理
- 分类与标签管理
- 教练管理
- 媒体资源管理
- 题目与考核内容管理
- 用户与会员管理
- 问答、评论与笔记管理
- 订单与退款管理
- 优惠券与营销活动管理
- 数据统计与运营看板

## 技术架构

悦动云采用前后端分离 + 微服务架构设计，围绕训练内容、互动社区、积分成长、营销活动、订单支付、用户中心等核心业务进行模块拆分。后端基于 Spring Boot 与 Spring Cloud Alibaba 构建，借助 Nacos 完成服务注册发现与配置管理，使用 Gateway 统一网关路由与接口鉴权，结合 Feign 实现微服务之间的远程调用；同时配合 Redis、Caffeine 提升热点数据访问性能，结合 MySQL、RabbitMQ、Zookeeper 等组件支撑业务数据存储、异步解耦与分布式协同，并通过 Docker 进行项目部署与环境管理。

## 业务模块

- 训练内容模块：负责训练信息、阶段目录、小节内容、训练详情等核心内容管理
- 跟练学习模块：负责视频学习、训练进度记录、跟练过程管理
- 互动社区模块：负责问答、回复、笔记等互动能力
- 积分成长模块：负责签到、积分记录、排行榜等成长激励能力
- 营销交易模块：负责优惠券、订单、支付、退款等交易流程
- 后台运营模块：负责训练、用户、内容、营销、订单等业务配置与运营管理

## 项目亮点

- 基于微服务拆分训练、用户、交易、营销、互动等核心业务模块
- 支持从训练浏览、下单购买、跟练学习到互动沉淀的完整业务闭环
- 引入积分、签到、排行榜、问答、笔记等机制提升用户活跃度
- 同时具备用户端与管理端，便于展示完整业务系统设计能力
- 适合作为健身训练平台与社区化内容平台的技术实践项目
